前提条件・免責（単価・thinking・その他）
  • Thinking: Gemini 公式の「Output (including thinking tokens)」などと同様、本計算でも thinking トークンは出力と同じ $/1Mで積み上げます。内部推論が大きいほど実費は伸びます。
  • 単価の前提: 各公式の Standard。Gemini 3.1 Pro 等は「200k トークン以下」、GPT-5.5 / GPT-5.4 系は「272k 未満コンテキスト」の掲載に準拠。長いプロンプトでは公式どおり単価が変わる場合があります。DeepSeek V4-Pro は 2026/05/31 より正式単価が 1/4 に改定（プロモーション価格が恒久化）。AWS Nova は Amazon Bedrock On-Demand 料金（Nova Micro が主要モデル中最安クラスの $0.035/$0.14）。
  • その他: Claude Fast mode やツール課金などは別レートのため、見積もりと実請求がずれることがあります。確定は各プロバイダの usage 表示を参照してください。
  • OpenAI・Anthropic は Cached input 等の別単価があります（本計算では未考慮）。
