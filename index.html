Thinking / 推論トークンについて

多くのモデルでは thinking（内部推論）も出力トークンと同じ $/1M で課金されます。 画面上の短文だけを入れると、実際の請求は桁違いになることがあります。 下のフォームでは「表示テキスト」と「thinking」を分けて入れられるようにし、 両方を足したトークン数に出力単価を掛けて計算します（API の usage と揃えるなら thinking を 0 にして、出力欄に合計だけ入れても構いません）。

前提条件・免責（単価・thinking・その他）
  • Thinking: Gemini 公式の「Output (including thinking tokens)」などと同様、本計算でも thinking トークンは出力と同じ $/1Mで積み上げます。内部推論が大きいほど実費は伸びます。
  • 単価の前提: 各公式の Standard。Gemini 3.1 Pro 等は「200k トークン以下」、GPT-5.4 系は「272k 未満コンテキスト」の掲載に準拠。長いプロンプトでは公式どおり単価が変わる場合があります。
  • その他: Claude Fast mode やツール課金などは別レートのため、見積もりと実請求がずれることがあります。確定は各プロバイダの usage 表示を参照してください。
  • OpenAI・Anthropic は Cached input 等の別単価があります（本計算では未考慮）。
